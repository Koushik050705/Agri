# Agri - Your All-in-One Agricultural Ecosystem

Agri is a modern, comprehensive agricultural platform designed to empower farmers with advanced technology and a seamless marketplace. From AI-driven crop diagnostics to a global marketplace, Agri provides the tools needed for sustainable and profitable farming.

## 🚀 Key Features

- **🌾 Farmer Marketplace**: A dedicated platform for farmers to list their crops directly for buyers, ensuring fair pricing and transparency.
- **🤖 Agri-AI Chatbot**: Real-time AI assistance for farming queries, pest management, and crop optimization.
- **🎙️ Voice Assistant**: Hands-free navigation and command execution, supporting multiple languages for accessibility.
- **🛰️ Agritech Tools**: Advanced tools for crop health monitoring, soil analysis, and yield prediction.
- **📅 Crop Calendar**: Personalized farming schedules based on regional data and crop types.
- **⚠️ Pest Alerts**: Early warning systems and tailored diagnostics for pest and disease control.
- **🌍 Global Reach**: Fully multilingual support (English, Hindi, etc.) using `i18next`.

## 🛠️ Tech Stack

- **Frontend**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling**: [Styled Components](https://styled-components.com/) & Vanilla CSS
- **Routing**: [React Router](https://reactrouter.com/)
- **State Management**: React Context API
- **AI/ML**: [TensorFlow.js](https://www.tensorflow.org/js)
- **Backend/Authentication**: [Supabase](https://supabase.com/)
- **Internationalization**: [i18next](https://www.i18next.com/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 📦 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16+)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Koushik050705/Agri.git
   cd Agri
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Setup environment variables:
   Create a `.env` file in the root and add your Supabase credentials:
   ```env
   VITE_SUPABASE_URL=your_supabase_url
   VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```
